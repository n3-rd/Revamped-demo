### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/n3-rd/VITE-TailwindCSS-Sass-boilerplate
   ```
1. Install NPM packages
   ```sh
   pnpm install
   ```
1. Run your dev server
   ```js
   pnpm start
   ```
